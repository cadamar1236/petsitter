# PetSitter 🐾

**Real-Time Pet Care You Can Trust**

Pet care marketplace connecting pet owners with vetted sitters, dog walkers, and groomers. GPS-tracked walks with photo updates, Stripe-powered instant booking, and vet emergency contacts on file. Revenue: 18% booking fee.

## Features

- **GPS-Tracked Walks** — Real-time map view of your pet's walk route with live location updates
- **Photo Updates** — Sitters share photos during the walk so you see your happy pet in real time
- **Instant Booking & Payment** — Stripe-powered secure payments with just one tap
- **Vetted Sitters** — Every sitter undergoes background checks, training, and community ratings
- **Vet Emergency Contacts** — Your vet's details are on file for every booking
- **Search & Filter** — Find sitters by name, location, or service category
- **Metrics Dashboard** — View total bookings, active sitters, and average ratings

## Tech Stack

- **Frontend:** React + Vite (JSX/TSX), Tailwind CSS, Lucide React Icons
- **Backend:** REST API with automatic retry logic and fallback data
- **Auth:** Local storage-based user session management
- **Deployment:** GitHub → CDN fallback auto-deploy

## Live Site

The app is live at: [https://petsitter.astarlabshub.com](https://petsitter.astarlabshub.com)

## Getting Started

1. Clone the repo
2. Run `npm install` in the project root
3. Start the dev server with `npm run dev`
4. Open `http://localhost:5173` in your browser
